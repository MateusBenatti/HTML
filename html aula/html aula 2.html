O que é HTML?

Linguagem de Marcação de Hipertexto

Hypetext Markup Language

O HTML é uma linguam de marcação representada por tags. As tags são interpretadas pelo navegador. A entrutura básica de um documento HTML é: html - tag principal; head - tag utilizada para delimitar o cabeçalho; title - tag onde colocamos o título da página; body - tag usada para definir o corpo do documento;
